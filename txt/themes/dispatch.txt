У нас много функций, которые меняют state,и  было бы неплохо вынести их куда нибудь в отдельный файл
Или еще куда


И вот можно вынести их в dispatch
Это будет функция, принимающая объект с параметром 'type'

И в зависимости от этого type будут выполняться те или иные действия


Вот пример: 


dispatch(action) { 
	if (action.type === 'ADD-POST'){
	 //	действия для создания поста
	} else if (action.type === '...'){
		...
	}
}

Вот такая структура

также можно не только передавать type

Если нам нужно еще и допустим value, то можем передать и его


Также стоит упомянуть, что метод объекта
это вот 
show(){

}
а функция 
show: () => {
	
}
