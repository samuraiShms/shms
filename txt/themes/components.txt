Компоненты



Компоненты - это маленькие фрагменты кода, которые можно так сказать, импортировать

Например есть базовый сайт в котором есть header main footer
Мы можем в основном файле написать каждый тег и заполнить его

Это будет выглядеть следующим образом

const App = () => {
	return(
		<header>
			header
		</header>
		<main>
			main
		</main>
		<footer>
			footer
		</footer>
	)
}


export default App;

Но если в каждом из блоков много контента, то будет не очень читабельно
Тут нам помогут компоненты, мы можем выносить фрагменты кода в отдельные файлы и импортировать их

Допустим у нас есть Header.jsx
В нём 
const Header = () => {
	<header>
	header
	</header>
}

export default Header;

После мы в App.jsx импортируем этот Header,а чтобы его использовать

то в App.jsx как тег вставляем компонент

const App = () => {
	return (
		<Header />
	)
}

export default App;

Также есть такие штуки, как пропсы, мы можем их передавать в компонент

const App = () => {
	const headerClass = 'header'
	return (
		<Header headerClass={headerClass} />
	)
}

const Header = ({headerClass}) => {
	<header className={headerClass.headerClass}>
	header
	</header>
}

Таким образом мы передали в компонент Header его класс, и когда мы получаем пропсы, то они получаются
Как объект, поэтому нам нужно их обвернуть в {} и при вызове указать headerClass.headerClass

или необязательно принимать пропс как название, которым мы передавали, мы можем его назвать как угодно, и вызывть по названию пропса
например props.headerClass

Если вкратце, то когда мы вызываем компонент и передаём какой либо атрибут, то полностью он, записывается в объект - пропс, то есть
вызываем компонент <Header hClass={'header'} /> и вот под  этим названием (hClass) в пропс и записывается hClass, с его значением, и оно запишется в пропс
А название пропса мы сами задаём в компоненте в аргументах функции
и в итоге получится такое:
название пропса = {
	название элемент пропса: его значение,
	...
}


Также в компоненту можно передать jsx как пропсы 

Для этого нужно компоненту вызвать как полноценный тег то есть так:

<Component>

</Component>

И внутрь открыть {}

И туда херячить jsx 

и потом из пропсов в компоненте получать children
