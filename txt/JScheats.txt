JS шпаргалки


1. Оператор spread
если нам нужно получить свойства масссива или объекта, то перед его названием надо поставить ...

пример: 
const array = [1, 2, 3]

console.log(...array) // 1 2 3

2. Объект и его свойства

Если в объекте ключ и значение одного имени

То можно просто указать название ключа, а значение автоматически поставится

пример:

let object = {
	name,
}

объект под капотом будет выглядеть так
let object = {
	name: name,
}

4. Замыкание

Замыкание - это функция, которая получает параметры, и возваращает другую функцию, в которой есть доступ к этим параметрам

пример

export const thunkCreator = (quantity) => { 
	return (dispatch) => {
		dispatch(setUsers(quantity)) // quantity
	}
}

или можем использовать стрелочный синатксис method = () => {}

6. Оператор rest


Есть функция 

и в неё попал объект через параметры
и если мы хотим достать из него что либо, то пишем название этого свойства объекта, и дальше
все остальные данные засовываем таким образом как ...rest и в ...rest уже не будет data 
и потом мы можем этим data манипулировать

const function = ({data, meta, ...rest}) => {
	console.log(data.name)
}
